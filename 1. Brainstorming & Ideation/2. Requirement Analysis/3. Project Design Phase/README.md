# Project Design Phase

## System Design
The Rain Prediction Project is designed with the following components:

1. **Dataset Input**
   - Collect weather/rainfall dataset
   - Preprocess the dataset

2. **Machine Learning Model**
   - Train the model using rainfall-related features
   - Save the trained model

3. **Web Application**
   - User enters input values
   - Flask app sends data to the model
   - Model predicts whether rain will occur or not

4. **Output Display**
   - Show prediction result on the web page

## Project Flow
Dataset → Preprocessing → Model Training → Flask App → User Input → Prediction Output
