# Project Planning Phase

## Project Plan
The Rain Prediction Project is planned in the following stages:

1. Dataset collection
2. Data preprocessing
3. Exploratory Data Analysis (EDA)
4. Model building
5. Model evaluation
6. Flask web app development
7. Deployment and GitHub documentation

## Timeline
- Week 1: Dataset collection and cleaning
- Week 2: EDA and preprocessing
- Week 3: Model training and testing
- Week 4: Flask integration and deployment
