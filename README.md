# Rain-Prediction-Project

## Project Overview
The **Rain Prediction Project** is a machine learning-based application developed to predict whether rainfall will occur based on weather-related input features.  
This project demonstrates the complete project lifecycle, including idea generation, requirement analysis, design, planning, development, testing, documentation, and demonstration.

---

## Objective
- To build a machine learning model for rain prediction
- To integrate the trained model with a Flask web application
- To provide a simple interface for users to enter weather details and get prediction results
- To organize the entire project in a structured GitHub repository
